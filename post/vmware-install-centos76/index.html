下载其中的 DVD版本用于安装：CentOS-7.6-x86_64-DVD-1810.iso

在vmware中安装centos

  1. 打开vmware,选择菜单: 文件/新建， 会弹出一个新建对话框【选择安装方式】；
  2. 在【选择安装方式】对话框中，将下载的CentOS-7.6-x86_64-DVD-1810.iso 拖到 【从光盘或映像中安装】中, 然后点击 【继续】键键；
  3. 点击【自定义设置】可以配置虚拟机使用的硬盘容易、内存、cpu、声音等硬件设备， 设置完后点击【完成】,进入后续的实际安装过程；
  4. 启动虚拟机，会出现如下图示安装页面，选择 (Install CentOS 7)，然后按 Enter键，进入安装流程;

  1. 选择安装语言 【简体中文】，然后进入(安装信息摘要)，在此界面可进行安装包配置、安装盘分配等操作， 我们都用默认的就可以了，如下图示:

  2. 点击【开始安装】，进入具体的安装过程，设置 root密码， 创建用户, 然后等待安装完成， 安装完成后有个【重启】按键，点击下就重启系统；

  3. 重启后进入系统，然后输入 用户名/密码，就可以进入系统使用了;
