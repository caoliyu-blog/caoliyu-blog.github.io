android获取屏幕分辨率的方法

· 2020-03-23 · # android

android可以通过DisplayMetrics来获取屏幕分辨率，dpi等参数

实现代码如下：

    DisplayMetrics dm = new DisplayMetrics(); 
    dm = this.getApplicationContext().getResources().getDisplayMetrics(); 
    int screenWidth = dm.widthPixels;       //实际像素宽 (px)
    int screenHeight = dm.heightPixels;    //实际像素高 (px)
    float density = dm.density;                //屏幕密度(每一个逻辑区域的px数)
    float dpi = dm.densityDpi;                // 屏幕的 dpi   (像素/英寸)
    int screenLogicWidth = dm.widthPixels/dm.density;      //逻辑宽
    int screenLogicHeight = dm.heightPixels/dm.density;    //逻辑高

各参数说明如下：
a. dm.widthPixels, dm.heightPixels 表示屏幕的实际 px (物理像素);
b. dm.densityDpi 表示屏幕的 dpi (像素/英寸);
c. density 屏幕密度， DPI/(160像素/英寸) 后得到的值;
dp = （DPI/（160像素/英寸））px = density px,
PX = density * DP
