一个功能强大的android图片加载开源库nostra13 / Android-Universal-Image-Loader

· 2020-03-24 · # android

它是早期广泛使用的开源图片加载库，强大又灵活的Android库, 用于加载,缓存,显示图片。它支持http，本地sd卡，assets及drawable等常用的存储方式显示，甚至还支持直接指定 .mp4文件名来加截视频的简略图片显示。

官方示例截图:

本开源项目的github地址为： https://github.com/nostra13/Android-Universal-Image-Loader
官方示例截图如下:

特点

a.多线程图像加载(支持同步/或异步加载);
b.各种自定义ImageLoader的配置（线程执行，下载，图片解析，内存/磁盘缓存，图片显示参数等）;
c.多种自定义图像显示功能(缓存，解码，显示选项支持);
d.图片内存/磁盘缓存支持(文件系统/或sd卡);
e.下载回调函数支持(含下载进度)

Android 2.0以上支持

官方使用示例说明：

依赖库加入

implementation 'com.nostra13.universalimageloader:universal-image-loader:1.9.5'

支持的url示例

"http://site.com/image.png" // from Web
"file:///mnt/sdcard/image.png" // from SD card
"file:///mnt/sdcard/video.mp4" // from SD card (video thumbnail)
"content://media/external/images/media/13" // from content provider
"content://media/external/video/media/13" // from content provider (video thumbnail)
"assets://image.png" // from assets
"drawable://" + R.drawable.img // from drawables (non-9patch images)

示例


// 获取单实例对象
ImageLoader imageLoader = ImageLoader.getInstance(); 

// 加载图片，解码到 bitmap并显示到 imageView中
imageLoader.displayImage(imageUri, imageView);

// 加载图片，解码到bitmap并回调加无完成
imageLoader.loadImage(imageUri, new SimpleImageLoadingListener() {
	@Override
	public void onLoadingComplete(String imageUri, View view, Bitmap loadedImage) {
		// Do whatever you want with Bitmap
	}
});

// 同步加载图片
Bitmap bmp = imageLoader.loadImageSync(imageUri);
